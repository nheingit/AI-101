  The read-only sheet property of the ProcessingInstruction interface
  contains the stylesheet associated to the ProcessingInstruction.

The xml-stylesheet processing instruction is used to associate a stylesheet in an XML file.ValueThe associated Stylesheet object, or null if there are none.Example<?xml version="1.0" encoding="UTF-8"?>
